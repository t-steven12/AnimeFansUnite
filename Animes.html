Animes

The Animes page regulates the Animes table and the First Episodes table.

This page shows all anime currently in the database along with their
associated information (including first episodes) in a table.
Animes can added by submitted the corresponding anime information.
Updating and/or deleting an anime is also possible. Adding, updating, and/or
deleting an anime also adds, updates, and/or deletes first episode information.
Users can search for a specific anime by submitting an anime title
and the Animes table will repopulate with the results.
