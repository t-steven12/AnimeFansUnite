Users

The Users page regulates the Users table.

This page shows a list of all current users of Anime Fans Unite as a table.
New users can be added by submitting a new user's first name and last name.
Users can be deleted. Deleting a User will removed all associated rows in
the Fav_animes composite table representing the M:M relationship between
Users and Animes.
