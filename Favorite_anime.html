Users' Favorited Animes

The Users' Favorited Animes page regulates the Fav_animes table.

This page displays a table of all animes favorited by a user.
A user can view a user's favorite animes by searching their user ID.
"User's Favorite Animes" will be populated by the search results.
A user can also favorite an anime which inserts the entered user ID and
anime ID into the Fav_animes table.
